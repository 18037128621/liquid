---
title: ceil
---

`ceil` rounds the input up to the nearest whole number.

| Input               | Output |
|--------------------:|:-------|
| `{ 1.2 | ceil }`    | 2      |
| `{ 1.7 | ceil }`    | 2      |
| `{ 2.0 | ceil }`    | 2      |
| `{ "18.3" | ceil }` | 19     |

It will attempt to cast any input to a number.
